基本CSS样式

Page actions 位置为页面主区域的最顶部

Page actions放到page-header的里面， 返回按钮在最左侧，页面的,其它的样式为btn，不允许使用其他的样式

  • 主要操作在最右侧样式为btn-primary
  • 其它按钮均为btn样式，并放在btn-group的容器里，数量 3 个以内， 按钮的文字应简短精确，适当的可以用tooltip补充说明
  • 返回按钮，不许更改文字及位置， 要跳转到哪个页面通过 tooltip， 并延迟 500毫秒 来提示

<div class="page-action">
    <button class="btn" tooltip-placement="bottom" tooltip="返回某个页面" tooltip-popup-delay='500'><i class="icon-arrow-left"></i></button>
    <button class="btn" tooltip-placement="bottom" tooltip="按钮解释说明" tooltip-popup-delay='500'>操作</button>
    <button class="btn" tooltip-placement="bottom" tooltip="按钮解释说明" tooltip-popup-delay='500'>操作</button>
    <button class="btn" tooltip-placement="bottom" tooltip="按钮解释说明" tooltip-popup-delay='500'>操作</button>
    <button class="btn btn-primary" tooltip-placement="bottom" tooltip="按钮解释说明" tooltip-popup-delay='500'>主要操作</button>
</div>

默认按钮

任何赋予.btn类的页面元素都会显示按钮样式。不过，通常是用于更好的表现<a> 和 <button> 页面元素。

按钮 class="" 描述
btn 带渐变的标准灰色按钮
btn btn-primary 提供额外的视觉感, 可在一系列的按钮中指出主要操作
btn btn-success 表示成功或积极的动作
btn btn-warning 提醒应该谨慎采取这个动作
btn btn-danger 表示这个动作危险或存在危险
